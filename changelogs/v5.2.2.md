## v5.2.2 - 安装链路加速与插件配置收口

### 安装链路
- **improve**: Linux / macOS / Windows 安装脚本改为优先从加速服务器拉取最新版本信息与安装包，下载失败后再回退 GitHub Release
- **improve**: README 安装命令改为默认使用加速服务器脚本地址，降低国内安装失败率

### 插件与通道
- **feat**: 飞书通道拆分为“飞书（官方版）”与“飞书（社区版）”两个独立入口，安装、启用、卸载分开管理
- **feat**: 插件安装优先走 OpenClaw 官方命令，失败后再回退兼容安装方式，并写入消息中心任务流
- **feat**: 插件卸载支持显式选择“仅卸载插件”或“卸载并清理通道配置”
- **improve**: 补充 wecom / dingtalk / qqbot 官方字段说明与更多可视化配置项
- **improve**: 通道页开始基于插件 `configSchema` 自动补充未手写的基础字段

### 稳定性
- **fix**: 卸载插件后按插件类型清理对应通道残留配置，飞书双版本保留互斥保护
- **fix**: 飞书双通道启用/禁用时显式映射到底层 `channels.feishu` 与正确插件 entry，避免状态错乱
