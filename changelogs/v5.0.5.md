# ClawPanel v5.0.5 更新日志

发布日期：2026-02-27

## 修复

### OpenClaw 一键安装自动安装 Node.js
- 修复全新服务器上点击「一键安装 OpenClaw」时因缺少 Node.js 直接报错退出的问题
- 安装脚本现在会自动检测并安装 Node.js v22 LTS（支持 apt/yum/dnf/brew）
- 自动配置 npm 淘宝镜像源加速下载
- 安装完成后自动初始化 OpenClaw 配置

### 安装权限修复
- 修复 OpenClaw 安装脚本在 Linux 上未使用 sudo 执行的问题
- 现在所有 Linux/macOS 安装任务统一使用 sudo 运行，确保系统包管理器和 npm 全局安装正常工作
