## v5.2.5 - Windows 环境检测与前置依赖可用性增强

### Windows 环境检测
- 扩展 Windows 常见运行时路径探测，补充 Node.js、npm、Git、Python、Chocolatey、Scoop、pyenv-win、ClawPanel npm-global 等目录
- Python 检测从仅识别 `python3` 改为依次尝试：`python3`、`python`、`py`
- 解决“先安装面板、后手动安装 Node.js/Git/Python，面板里仍显示未安装”的高频误判问题

### OpenClaw 安装前校验
- Windows / macOS 安装 OpenClaw 前，除了检查 `node` 与 `git`，还会同步确认 `npm` 可用
- 避免出现“检测到 Node.js，但 npm 实际不可用，安装仍然半途失败”的情况

### 稳定性
- 补充 Windows 路径覆盖与 Python / npm 回退相关测试
