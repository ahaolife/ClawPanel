## v5.0.26 - 稳定性修复（跨环境一致性）

### Bug 修复
- **fix**: 修复版本管理页“当前版本”可能与 `openclaw --version` 不一致的问题（优先使用实时 CLI 版本，解决 Issue #23）
- **fix**: 修复 macOS 下 NapCat 诊断/监控误报 Docker 未安装，统一 Docker 命令执行环境与架构兼容
- **fix**: 修复 OpenClaw 网关在服务环境下因 `PATH/HOME` 缺失导致退出（`exit code 127/1`）
- **fix**: 修复 NapCat 已运行但 `channels.qq.wsUrl` 未注入导致“收到消息不回复”的问题
- **fix**: 修复 QQ 插件目录属主不匹配被 OpenClaw 安全策略拦截（`unknown channel id: qq`）

### 兼容性改进
- **improve**: 增强系统环境采集在 macOS 的字段完整性（发行版、内存、CPU、负载、运行时长）
- **improve**: 增强 Node/npm 与 Docker 在服务态下的检测健壮性（PATH/HOME、二进制路径、macOS 架构回退）
